# Human Auto Typing

Chrome/Edge extension for natural-looking typing on web inputs.

## Features

- Variable typing speed
- Random nearby-key typos with correction
- Human-like pauses around spaces and punctuation

## Install

1. Open the browser extensions page.
2. Enable Developer mode.
3. Load the unpacked folder in this directory.

## Use

1. Click into an input, textarea, or editable element on a web page.
2. Open the extension popup.
3. Enter text, adjust speed and typo rate, then start typing.
